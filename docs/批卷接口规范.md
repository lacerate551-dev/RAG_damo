# 批卷接口规范

> **版本**: v1.0
> **创建日期**: 2026-04-12
> **状态**: 待前后端确认

---

## 概述

本文档定义了出题系统和批卷系统的数据格式规范，供前后端开发参考对接。

---

## 一、出题输出格式（后端 → 前端）

### 1.1 完整结构

```json
{
  "exam_id": "uuid-xxxx-xxxx",
  "source_file": {
    "path": "public/产品手册.pdf",
    "collection": "public_kb"
  },
  "choice_questions": [...],
  "blank_questions": [...],
  "short_answer_questions": [...],
  "total_count": 9,
  "total_score": 22,
  "generated_at": "2026-04-12T12:00:00",
  "created_by": "user_001"
}
```

### 1.2 选择题格式

```json
{
  "id": "q_choice_001",
  "content": "根据保密制度，公司最高机密的处理原则是什么？",
  "options": [
    "A. 可向客户透露",
    "B. 严禁外传",
    "C. 部门内共享",
    "D. 仅领导知晓"
  ],
  "answer": "B",
  "analysis": "根据保密制度第1条规定，公司最高机密严禁外传，仅限特定人员知晓。",
  "knowledge_points": ["保密制度", "信息安全"],
  "difficulty": 2,
  "score": 2,
  "source_file": "public/产品手册.pdf",
  "source_snippet": "原文相关片段..."
}
```

**字段说明**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | ✅ | 题目唯一标识 |
| `content` | string | ✅ | 题干内容 |
| `options` | array | ✅ | 选项列表，格式为 `["A. 选项内容", ...]` |
| `answer` | string | ✅ | 正确答案，单个字母（如 "A", "B"） |
| `analysis` | string | ✅ | 答案解析 |
| `knowledge_points` | array | ❌ | 知识点标签 |
| `difficulty` | int | ❌ | 难度等级 1-5，默认 3 |
| `score` | int | ✅ | 题目分值 |
| `source_file` | string | ❌ | 来源文件路径 |
| `source_snippet` | string | ❌ | 来源文本片段 |

### 1.3 填空题格式

```json
{
  "id": "q_blank_001",
  "content": "公司财务报表应在每季度结束后______天内提交。",
  "answer": "15",
  "analysis": "根据财务管理制度第5条规定，季度报表需在季后15天内提交。",
  "knowledge_points": ["财务管理"],
  "difficulty": 3,
  "score": 3
}
```

**字段说明**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | ✅ | 题目唯一标识 |
| `content` | string | ✅ | 题干内容，空缺处用 `______` 表示 |
| `answer` | string | ✅ | 正确答案 |
| `analysis` | string | ✅ | 答案解析 |
| `knowledge_points` | array | ❌ | 知识点标签 |
| `difficulty` | int | ❌ | 难度等级 1-5 |
| `score` | int | ✅ | 题目分值 |

### 1.4 简答题格式

```json
{
  "id": "q_short_001",
  "content": "简述公司数据安全的三道防线。",
  "reference_answer": {
    "points": [
      {"point": "技术防线（防火墙、加密、访问控制等）", "score": 3},
      {"point": "制度防线（安全规定、审批流程、应急预案）", "score": 3},
      {"point": "人员防线（安全培训、意识教育、考核机制）", "score": 4}
    ],
    "total_score": 10
  },
  "analysis": "评分要点说明...",
  "knowledge_points": ["数据安全"],
  "difficulty": 4,
  "score": 10
}
```

**字段说明**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | string | ✅ | 题目唯一标识 |
| `content` | string | ✅ | 题干内容 |
| `reference_answer` | object | ✅ | 参考答案，包含评分要点 |
| `reference_answer.points` | array | ✅ | 得分点列表 |
| `reference_answer.points[].point` | string | ✅ | 得分点描述 |
| `reference_answer.points[].score` | int | ✅ | 该得分点分值 |
| `analysis` | string | ❌ | 整体解析 |
| `knowledge_points` | array | ❌ | 知识点标签 |
| `difficulty` | int | ❌ | 难度等级 1-5 |
| `score` | int | ✅ | 题目总分值 |

## 二、批卷输入格式（前端 → 后端）

### 2.1 完整结构

```json
{
  "exam_id": "uuid-xxxx-xxxx",
  "student_id": "STU_2023001",
  "student_name": "张三",
  "answers": [
    {
      "question_id": "q_choice_001",
      "question_type": "choice",
      "question_content": "根据保密制度，公司最高机密的处理原则是什么？",
      "options": ["A. 可向客户透露", "B. 严禁外传", "C. 部门内共享", "D. 仅领导知晓"],
      "correct_answer": "B",
      "max_score": 2,
      "student_answer": "B"
    },
    {
      "question_id": "q_blank_001",
      "question_type": "blank",
      "question_content": "公司财务报表应在每季度结束后______天内提交。",
      "correct_answer": "15",
      "max_score": 3,
      "student_answer": "10"
    },
    {
      "question_id": "q_short_001",
      "question_type": "short_answer",
      "question_content": "简述公司数据安全的三道防线。",
      "correct_answer": "{\"points\":[{\"point\":\"技术防线\",\"score\":3},{\"point\":\"制度防线\",\"score\":3},{\"point\":\"人员防线\",\"score\":4}]}",
      "max_score": 10,
      "student_answer": "第一道是技术防护，包括防火墙和加密；第二道是制度管理；第三道是员工培训。"
    }
  ]
}
```

### 2.2 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `exam_id` | string | ✅ | 试卷ID |
| `student_id` | string | ✅ | 学生学号 |
| `student_name` | string | ❌ | 学生姓名，默认"匿名" |
| `answers` | array | ✅ | 答案列表 |

### 2.3 答案项格式

**选择题**：
```json
{
  "question_id": "q_choice_001",
  "question_type": "choice",
  "question_content": "题干内容",
  "options": ["A. ...", "B. ...", "C. ...", "D. ..."],
  "correct_answer": "B",
  "max_score": 2,
  "student_answer": "A"
}
```

**填空题**：
```json
{
  "question_id": "q_blank_001",
  "question_type": "blank",
  "question_content": "题干内容______",
  "correct_answer": "正确答案",
  "max_score": 3,
  "student_answer": "学生答案"
}
```

**简答题**：
```json
{
  "question_id": "q_short_001",
  "question_type": "short_answer",
  "question_content": "题干内容",
  "correct_answer": "{\"points\":[{\"point\":\"要点1\",\"score\":3}]}",
  "max_score": 10,
  "student_answer": "学生作答内容..."
}
```

**注意**：简答题的 `correct_answer` 是 JSON 字符串格式，需要解析后使用。

## 三、批卷输出格式（后端 → 前端）

### 3.1 完整结构

```json
{
  "report_id": "report-uuid-xxxx",
  "exam_id": "uuid-xxxx-xxxx",
  "student_id": "STU_2023001",
  "student_name": "张三",
  "total_score": 12,
  "max_score": 15,
  "score_rate": 80.0,
  "graded_at": "2026-04-12T14:30:00",
  "results": [...],
  "summary": {
    "strengths": ["选择题掌握较好", "简答题要点覆盖全面"],
    "weaknesses": ["填空题记忆不准确"],
    "recommendations": ["重点复习财务管理制度第3章"]
  }
}
```

### 3.2 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `report_id` | string | ✅ | 批阅报告ID |
| `exam_id` | string | ✅ | 试卷ID |
| `student_id` | string | ✅ | 学生学号 |
| `student_name` | string | ✅ | 学生姓名 |
| `total_score` | float | ✅ | 总得分 |
| `max_score` | int | ✅ | 满分 |
| `score_rate` | float | ✅ | 得分率（百分比） |
| `graded_at` | string | ✅ | 批阅时间（ISO格式） |
| `results` | array | ✅ | 各题批阅结果 |
| `summary` | object | ❌ | 整体评价摘要 |

### 3.3 批阅结果格式

**选择题结果**：
```json
{
  "question_id": "q_choice_001",
  "question_type": "choice",
  "correct": true,
  "score": 2,
  "max_score": 2,
  "student_answer": "B",
  "correct_answer": "B"
}
```

**填空题结果**：
```json
{
  "question_id": "q_blank_001",
  "question_type": "blank",
  "correct": false,
  "score": 0,
  "max_score": 3,
  "student_answer": "10",
  "correct_answer": "15",
  "feedback": "正确答案是15天，请复习财务管理制度。"
}
```

**简答题结果**：
```json
{
  "question_id": "q_short_001",
  "question_type": "short_answer",
  "score": 8,
  "max_score": 10,
  "student_answer": "第一道是技术防护...",
  "score_details": [
    {"point": "技术防线", "earned": 3, "max": 3},
    {"point": "制度防线", "earned": 2, "max": 3},
    {"point": "人员防线", "earned": 3, "max": 4}
  ],
  "feedback": "整体回答较好，制度防线描述不够具体。",
  "highlights": ["技术防线表述准确"],
  "shortcomings": ["制度防线未具体说明"],
  "suggestions": ["建议补充具体的制度名称"]
}
```

### 3.4 简答题结果字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `score_details` | array | ❌ | 各得分点详情 |
| `score_details[].point` | string | ✅ | 得分点描述 |
| `score_details[].earned` | int | ✅ | 实得分数 |
| `score_details[].max` | int | ✅ | 该点满分 |
| `feedback` | string | ❌ | 整体评语 |
| `highlights` | array | ❌ | 答案亮点 |
| `shortcomings` | array | ❌ | 不足之处 |
| `suggestions` | array | ❌ | 改进建议 |

## 四、API 端点

### 4.1 出题接口

**请求**：
```http
POST /exam/generate-by-file
Content-Type: application/json
X-User-ID: user_001
X-User-Role: admin
X-User-Department: general

{
  "file_path": "public/产品手册.pdf",
  "collection": "public_kb",
  "choice_count": 5,
  "blank_count": 2,
  "short_answer_count": 2,
  "difficulty": 3,
  "choice_score": 2,
  "blank_score": 3
}
```

**响应**：参见 [一、出题输出格式](#一出题输出格式后端--前端)

### 4.2 批卷接口

**请求**：
```http
POST /exam/grade-from-mysql
Content-Type: application/json
X-User-ID: teacher_001
X-User-Role: admin

{
  "exam_id": "uuid-xxxx-xxxx",
  "student_id": "STU_2023001",
  "student_name": "张三",
  "answers": [...]
}
```

**响应**：参见 [三、批卷输出格式](#三批卷输出格式后端--前端)

## 五、错误处理

### 5.1 错误响应格式

```json
{
  "error": "错误类型",
  "message": "详细错误信息",
  "details": {}
}
```

### 5.2 常见错误码

| HTTP状态码 | 错误类型 | 说明 |
|-----------|---------|------|
| 400 | bad_request | 请求参数格式错误 |
| 401 | unauthorized | 未认证 |
| 403 | forbidden | 权限不足 |
| 404 | not_found | 资源不存在 |
| 500 | internal_error | 服务器内部错误 |

## 七、测试文档生成方法

测试文档通过脚本 `generate_test_docs.py` 自动生成，支持以下格式：

- **TXT**: 直接文本写入
- **DOCX**: 使用 `python-docx` 库生成
- **XLSX**: 使用 `openpyxl` 库生成
- **PDF**: 使用 `reportlab` 库生成

运行命令：
```bash
python generate_test_docs.py
```

依赖安装：
```bash
pip install python-docx openpyxl reportlab
```
